VectorHire

ATS inteligente com RAG Pipeline

Sistema completo de recrutamento que cruza currículos com vagas usando Retrieval-Augmented Generation. Extrai PDFs com PyMuPDF, fatia com Semantic Chunking, vetoriza localmente com sentence-transformers e busca similaridade via PostgreSQL + pgvector. O Gemini 2.5 Flash gera pareceres estruturados (Pydantic) para cada candidato. API assíncrona com FastAPI, persistência via SQLAlchemy (Repository Pattern) e frontend desacoplado.

Stream-Guard-Kafka

Detecção de fraudes em tempo real

Pipeline de Arquitetura Kappa com processamento híbrido: ksqlDB aplica Tumbling Windows para detectar alta velocidade transacional, enquanto Python processa regras de negócio complexas (blacklist, contexto). Kafka 3.6 em KRaft mode (sem ZooKeeper), dados sintéticos realistas via Faker, contratos tipados com Pydantic e persistência em PostgreSQL via SQLAlchemy. Infraestrutura inteira orquestrada com Docker Compose.

BQ Schema Migrator

Governança de schemas no BigQuery

Ferramenta de migração inspirada em Flyway/Liquibase, feita para BigQuery. Cada script SQL roda exatamente uma vez, com controle de idempotência via checksums SHA256 e tabela de controle. Scripts com header @scheduled são deployados automaticamente como Scheduled Queries pela Data Transfer API. Suporte a placeholders (${PROJECT}, ${DATASET}), auto-discovery de credenciais e testes determinísticos com Pytest.

GCP BigQuery Data Transfer API Python Pytest SHA256 CI/CD

What-Price

Pipeline ELT de mercado cripto

Pipeline end-to-end que extrai dados de mercado das top 20 criptomoedas via CoinGecko API a cada hora. Orquestrado com Apache Airflow 3 (TaskFlow API + Astro), carrega em PostgreSQL Serverless (Neon) com ingestão idempotente, e expõe métricas em um dashboard Streamlit com Plotly: dominância de mercado, índice de performance, volatilidade 24h e distância do ATH. Arquitetura Bronze/Silver inspirada na Medalhão.
