BQ Schema Migrator

Governança de schemas no BigQuery

Ferramenta de migração inspirada em Flyway/Liquibase, feita para BigQuery. Cada script SQL roda exatamente uma vez, com controle de idempotência via checksums SHA256 e tabela de controle. Scripts com header @scheduled são deployados automaticamente como Scheduled Queries pela Data Transfer API. Suporte a placeholders (${PROJECT}, ${DATASET}), auto-discovery de credenciais e testes determinísticos com Pytest.
