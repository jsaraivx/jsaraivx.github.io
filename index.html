Sobre

Me chamo João Saraiva. Sou Engenheiro de Dados na Avdata Consulting, onde lidero a modernização de fluxos de dados críticos para clientes globais em operação remota.

Antes de dados, passei 3 anos construindo software (React, Node, TypeScript). Essa base me deu algo que puro SQL não dá: visão sobre como os dados impactam o produto que o usuário toca. Hoje, aplico isso projetando arquiteturas no ecossistema Google Cloud Platform — com foco em Data Quality, otimização de custos (FinOps) e pipelines idempotentes que os times de negócio confiam para tomar decisões.

  • Orquestração resiliente com Airflow e Cloud Composer
